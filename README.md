# SQL Assignments

This repository contains SQL assignments completed as part of my Database Management System (DBMS) practice.

## Repository Contents

- **Assignment1.sql**
  - Database creation
  - Table creation
  - Data insertion
  - Basic SQL queries (SELECT, WHERE, ORDER BY)

- **Assignment2.sql**
  - ALTER TABLE operations
  - Aggregate functions
  - GROUP BY and HAVING
  - Advanced SQL queries

- **Assignment 3.sql**
  - SQL queries covering advanced database concepts and problem-solving exercises.
  - Includes queries using joins, subqueries, aggregate functions, grouping, filtering, and other SQL operations as required in the assignment.

## Software Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

## How to Run

1. Open SQL Server Management Studio (SSMS).
2. Connect to your SQL Server instance.
3. Open the required `.sql` file.
4. Execute the script by pressing **F5** or clicking **Execute**.
