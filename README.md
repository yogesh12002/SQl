# SQL Assignments

This repository contains SQL assignments completed as part of my database learning and practice.

## Repository Contents

- **Assignment1.sql** – SQL queries covering basic database operations such as:
  - Database creation
  - Table creation
  - Data insertion
  - Basic SQL queries

- **Assignment2.sql** – SQL queries covering advanced concepts such as:
  - Joins
  - Aggregate functions
  - GROUP BY and HAVING
  - Subqueries
  - Other SQL operations

## Requirements

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

## How to Run

1. Open SQL Server Management Studio (SSMS).
2. Connect to your SQL Server instance.
3. Open the required `.sql` file.
4. Execute the script using the **Execute** button or press **F5**.
